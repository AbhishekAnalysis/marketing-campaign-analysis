
#  Marketing Campaign Performance Analysis

A end-to-end data cleaning and analysis project on messy marketing campaign data
across 6 channels — built to demonstrate real-world data wrangling skills.

---

##  Problem Statement

Raw campaign data had inconsistent channel names, reversed dates, mixed currency
formats, incorrect active flags, and missing values — making it unusable for analysis.

---

## 💡 Key Insights (Post Cleaning)

- **6 channels** tracked: Facebook, Google Ads, Instagram, TikTok, Email, Unknown
- **Highest revenue per conversion** → Google Ads (₹80)
- **Lowest revenue per conversion** → TikTok (₹30)
- Dates corrected for **logically reversed** start/end entries
- Spend standardized by removing currency symbols and flipping negatives
